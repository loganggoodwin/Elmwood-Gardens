# Elmwood Gardens Tribute Project

A heartfelt tribute project celebrating **Elmwood Gardens in Manchester, New Hampshire**, honoring the families who grew up there from the **1970s through the 1990s**, the people who are still there, and those who have moved on.

This package is ready to upload to **GitHub** as a simple media-and-artwork repository.

---

## Project Description

This repository contains:

- A finished **cover / album-style artwork image** for the Elmwood Gardens tribute
- An **MP3 audio file** of the song
- An **MP4 video file** for sharing or playback
- A short project structure that is easy to upload and organize on GitHub

The artwork includes the message:

> For all the families who grew up here,  
> for those still here, and for those  
> who have moved on —  
> *Elmwood Gardens* will always live in our hearts.

## How to Upload to GitHub

1. Create a new repository on GitHub.
2. Name it something like:
   - `elmwood-gardens-tribute`
   - `elmwood-gardens-song`
   - `elmwood-gardens-memory-project`
3. Upload all files from this folder into your repository.
4. Make sure `README.md` is included in the root of the repo.
5. Commit the files.
